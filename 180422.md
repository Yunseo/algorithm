### 1.3 urlify
- 문제에서 'in place' 알고리즘을 짜보기 => idea: array의 역순으로 채워나가보자. 
  ```java
  void urlify (char[] charArr, int trueLenth) {
    int urlIdx = charArr.length - 1;
    for(int i = trueLength - 1; i > -1; i--) {
      if(charArr[i] == ' ') {
        charArr[urlIdx--] = '0';
        charArr[urlIdx--] = '2';
        charArr[urlIdx--] = '%';
      } else {
        charArr[urlIdx--] = char[i];
      }
    }
  }
  ```
